IAM & Microsoft Cloud Engineering Portfolio

Welcome. This portfolio documents hands-on Microsoft identity, endpoint, automation, and governance projects built around a realistic business scenario: SweetCrust Bakery Ltd.

The projects demonstrate how a small business can be modernized using enterprise-grade Microsoft technologies including Active Directory, Microsoft Entra ID, Microsoft Intune, Privileged Identity Management, Access Reviews, and Microsoft Graph PowerShell automation.

Built a staged identity modernization project for SweetCrust Bakery Ltd, beginning with Windows Server and Active Directory Domain Services, then extending identity into Microsoft Entra ID using Entra Connect, and finally integrating cloud-managed endpoints through Intune.

Skill Focus: Windows Server, AD DS, DNS, Group Policy, Entra Connect, Entra ID, Intune

Role Alignment: Systems Administrator | Hybrid Cloud Administrator | Microsoft 365 Administrator
